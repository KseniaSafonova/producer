# BaseButton

Компонент кнопки

## theme

Тема кнопки

- dark (default)
- light
- accent

## size

Размер кнопки

- small
- middle (default)
- large

## isFullWidth

Кнопка растягивающаяся на всю ширину блока

## href

Cсылка на внешнюю страницу

## to

Ссылка на якорь внутри страницы

```
:to="{ name: 'PageLanding', hash: '#connect' }"
```
